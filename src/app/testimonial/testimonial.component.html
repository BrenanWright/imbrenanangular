These testimonials were then stored in their own table in the MySQL Database and I approved them via database updates on the backend website. When one visited this page it would load the data as a JSON file and parse through each response and build a card below. This was dynamic and would update whenever the database had a change submitted. If you're curious to see the code this is the Github link!
